# Icon Placeholders

To complete the extension setup, you need to add PNG icon files:

Required files:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)  
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

You can:
1. Use the icon.svg file as a base and convert to PNG
2. Create custom icons matching the Media Buyer's Toolkit theme
3. Use online SVG to PNG converters
4. Use design tools like Figma, Canva, or Photoshop

Recommended colors:
- Primary: #667eea (blue)
- Secondary: #764ba2 (purple)
- Background: White or transparent
- Icon style: Modern, clean, professional

The icons should represent media buying/analytics (charts, graphs, targeting symbols).
